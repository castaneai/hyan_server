# HyanServer

## Requirements

- Elixir >= 1.4
- MySQL server >= 5.6 (user: root, password: "")

## Install

```
mix deps.get
mix ecto.create
mix ecto.migrate
```

## Start

```
mix phoenix.server
```

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

## Test

```
mix test
```
